# Number Guessing Game

Welcome to the Number Guessing Game! This project is a simple Python-based game where players try to guess a randomly generated number within a certain range.

## Table of Contents
- [Installation](#installation)
- [Usage](#usage)
- [Game Rules](#game-rules)
- [Contributing](#contributing)
- [License](#license)

## Installation

To get started, clone the repository to your local machine:

```
git clone https://github.com/yourusername/number-guessing.git
```

Navigate into the project directory:

```
cd number-guessing
```

Make sure you have Python installed on your machine. You can download it from [python.org](https://www.python.org/downloads/).

## Usage

Run the game by executing the following command in your terminal:

```
python number_guess.py
```

Follow the on-screen instructions to play the game.

## Game Rules

1. The game will randomly select a number within a specified range.
2. You will have a limited number of attempts to guess the number.
3. After each guess, you will receive feedback indicating whether your guess was too high, too low, or correct.
4. Try to guess the number in as few attempts as possible!
